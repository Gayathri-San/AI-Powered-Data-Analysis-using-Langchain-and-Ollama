## **AI POWERED DATA ANALYSIS USING LANGCHAIN AND OLLAMA**

### **INTRODUCTION**
- Data analysis is essential for extracting useful insights from raw data 
- Traditional methods require programming knowledge like Python or SQL 
- Non-technical users face difficulty in analyzing and visualizing data 
- Artificial Intelligence enables automation of data analysis tasks 
- This project uses LangChain and Ollama to understand natural language queries 
- It converts user queries into structured data operations and generates insights 
- The system provides an easy and interactive way to analyze datasets

### **OBJECTIVE**
- Enable users to analyze data using natural language queries 
- Convert text input into structured data analysis operations 
- Generate automatic charts and visualizations from datasets
- Provide AI-generated insights from analyzed data 
- Integrate LangChain and Ollama for intelligent query processing 
- Create an interactive and user-friendly analytics interface

 ### **TECHNOLOGIES USED**
- LangChain
- Ollama
- Python
- Matplotlib 
- Streamlit

 ### **SYSTEM ARCHITECTURE**
 - User uploads dataset and enters natural language query through Streamlit interface 
 - Query is sent to LangChain for processing and workflow management 
 - Ollama (LLM) converts the query into structured JSON format 
 - Python processes the JSON and performs data analysis operations 
 - Result table is generated from processed data 
 - Charts are created using Matplotlib for visualization 
 - Ollama generates meaningful insights from analyzed results 
 - Final output result such as table , charts and insights are displayed in Streamlit UI
